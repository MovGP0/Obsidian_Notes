## Installation
```powershell
winget install nodejs
winget install yarn
yarn global add @angular/cli
```

## Examples

Create Project
```powershell
ng new PROJECTNAME
```

Add Component to Project
```powershell
cd PROJECTNAME
ng generate component COMPONENTNAME
```

Run unit tests
```powershell
ng test
```

Run lint
```powershell
ng lint
```

Run project
```powershell
ng serve
```
