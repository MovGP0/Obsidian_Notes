[[Rust]] package manager.

## Examples

Get version info
```bash
cargo --version
```

Get help
```powershell
cargo --help
cargo COMMAND --help
```

Create a new project
```powershell
cargo new 'PROJECTNAME'
```

Check program
```powershell
cargo check
```

Run tests
```powershell
cargo test
```

Run benchmarks
```powershell
cargo bench
```

Build program
```powershell
cargo build # -dev version
cargo build -dev
cargo build -release
cargo build -test
cargo build -bench
```

Build and run program
```powershell
cargo run
```

Clean build
```powershell
cargo clean
```
