CLI client for [[Kubernetes]] REST API.

### Help

| Command                  | Description                      |
| ------------------------ | -------------------------------- |
| `kubectl`                | gets help                        |
| `kubectl help`           | gets help                        |
| `kubectl --help`         | gets help                        |
| `kubectl COMMAND --help` | gets help for a specific command |
| `kubectl options`        | lists command line parameters    |
